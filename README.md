# 🎯 團康計分器

一個專為手機設計的團康活動計分應用，支援多隊伍計分、即時排名顯示。

## ✨ 功能特色

- 📱 **移動端優化**：專為手機瀏覽器設計，觸控友好
- 🎨 **美觀界面**：現代化漸層設計，視覺效果佳
- 👥 **多隊伍支援**：預設4個隊伍，可新增/刪除隊伍
- 📊 **即時排名**：自動根據分數排序顯示排名
- ➕➖ **靈活計分**：支援 +1、+5、-1、-5 分數調整
- 🔄 **重置功能**：可重置單一隊伍或全部隊伍分數
- 💾 **本地儲存**：分數會自動保存（可選功能）

## 🚀 快速開始

### 安裝依賴

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

應用會在 `http://localhost:3000` 啟動

### 建置生產版本

```bash
npm run build
```

## 📱 使用方式

1. **啟動應用**：在手機瀏覽器中打開應用網址
2. **計分操作**：
   - 點擊 `+1` 或 `+5` 按鈕增加分數
   - 點擊 `-1` 或 `-5` 按鈕減少分數
   - 點擊「重置分數」將該隊伍分數歸零
3. **管理隊伍**：
   - 點擊「+ 新增隊伍」添加新隊伍
   - 點擊隊伍卡片右上角的 `×` 刪除隊伍
4. **重置全部**：點擊右上角「重置全部」按鈕將所有隊伍分數歸零

## 🎨 設計特點

- 每個隊伍有獨特的顏色標識
- 大號分數顯示，方便遠距離查看
- 觸控按鈕設計，適合手指操作
- 流暢的動畫效果，提升使用體驗
